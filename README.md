# farkle
A web application for the dice game Farkle

## Rules and Objective

Farkle is a "push-your-luck" dice game where players roll six dice to score points through combinations like straights, three-of-a-kind, or5s (50) and 1s (100). Players must set aside at least one scoring die per roll. If no dice score, it’s a "Farkle," losing all points accumulated that turn. After scoring, players may bank points or risk them by rolling remaining dice. The first player to reach 10,000 points wins, after a final round
